# Booking Management System

Complete booking management for businesses including application viewing, worker profile cards with reliability scores, accept/reject functionality, bulk actions, worker notes, and real-time updates via Supabase Realtime.

## Rationale
Businesses need efficient tools to manage applications and make hiring decisions. Reliability scores and worker profiles address the no-show and quality concerns from social media hiring.

## User Stories
- As a hotel manager, I want to see worker reliability scores so that I can hire dependable staff
- As a business, I want to accept multiple workers at once so that I can fill large shifts quickly
- As a business, I want to add notes to workers so that I can remember good performers

## Acceptance Criteria
- [ ] Businesses can view all applications for their jobs
- [ ] Worker profile cards show reliability score, skills, and experience
- [ ] Accept/reject actions update booking status
- [ ] Bulk actions allow accepting or rejecting multiple workers
- [ ] Worker notes can be added for reference
- [ ] Real-time updates show new applications immediately
- [ ] Booking status changes trigger notifications to workers
