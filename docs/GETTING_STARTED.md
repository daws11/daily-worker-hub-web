## 🏗️ How the Platform Works

### For Businesses

```
Create Account → Complete Profile → Post Job → Review Applications → Hire Workers → Pay via Wallet
```

1. **Create Account** — Register as a business user
2. **Complete Profile** — Add company information and verification documents
3. **Post a Job** — Describe the daily work you need (title, skills required, wage, schedule)
4. **Review Applications** — Browse worker profiles and applications
5. **Hire Workers** — Select and assign workers to your job
6. **Track Attendance** — Use check-in/check-out to verify work hours
7. **Pay via Wallet** — Transfer payment through the in-app wallet system

### For Workers

```
Create Account → Complete Profile → Browse Jobs → Apply → Get Hired → Check In/Out → Get Paid
```

1. **Create Account** — Register as a worker
2. **Complete Profile** — Add your skills, experience, and location
3. **Browse Jobs** — Explore available daily work opportunities
4. **Apply** — Submit applications for jobs that match your skills
5. **Get Hired** — Receive assignment confirmation from businesses
6. **Check In/Out** — Record your attendance at the job site
7. **Get Paid** — Receive payment directly to your wallet

## 🔑 Key Features at a Glance

- **💳 Wallet System** — Top-up balance and pay workers (or receive earnings) via QRIS, Virtual Account, or GoPay
- **⭐ 2-Way Reviews** — Both businesses and workers rate each other after each job
- **🏆 Badge System** — Skill verification and reliability badges build trust
- **💬 Interview System** — Chat and voice call to coordinate before hiring
- **📅 Booking Management** — Full lifecycle from posting to completion with check-in/check-out
- **📱 Push Notifications** — Real-time updates on applications, hires, and payments
